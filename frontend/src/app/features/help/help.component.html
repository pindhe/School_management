How to use the admin panel

  1. Dashboard — live counts from database.
  2. Students — CRUD on student table.
  3. Reports — analytics from DB (Admin).
  4. Users — list from app_user table (Admin).
  5. Settings — saved in user_settings table.
  6. Profile — your account from database.
